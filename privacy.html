Effective Date: April 14, 2026

This application is a personal desktop tool used to upload videos to social media accounts through official platform authorization flows.

1. Information We Access

This application may access the following information:

  • Authorization tokens provided by supported platforms such as TikTok and YouTube
  • User-selected local video files for upload
  • Basic account permissions explicitly granted by the user

2. How We Use Information

The information accessed is used only for the following purposes:

  • To authenticate the user with connected social media platforms
  • To upload videos selected by the user
  • To maintain session continuity and improve user experience

3. Data Storage

This application stores authorization tokens locally on the user’s device. Video files remain on the user’s local device and are only accessed when the user explicitly selects content for upload.

4. Data Sharing

This application does not sell, rent, or share user data with third parties. Data is only transmitted to official platform APIs required to complete user-requested actions.

5. Data Retention

Locally stored tokens may remain on the device until they expire, are revoked, or are removed by the user by clearing local application data or uninstalling the application.

6. Security

Reasonable efforts are made to protect locally stored data and to use official, secure authorization methods provided by third-party platforms.

7. User Control

Users may revoke access at any time by:

  • Removing the app’s access from the connected platform account settings
  • Deleting local application data
  • Uninstalling the application
